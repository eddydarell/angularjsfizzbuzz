FizzBuzz

Description

FizzBuzz is a counting game is which 3 numbers are chosen.
2 of which are dividers and the last is the max number to count till.
The game starts by counting from 1 to the max number to count till and everytime the counted number is dividable by the smallest divider we display / say "Fizz" instead of the digit.
We do the same with the greatest divider but then we say "Buzz" instead.
If the current number is dividable by both dividers, we say / display "FizzBuzz".

You can try this yourself in the form on the right.
the results are updated as you change the value of the fields. If some values are missing the default values for the dividers are 3 and 4 and 100 for the max count.
